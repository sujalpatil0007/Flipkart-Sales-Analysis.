# 📊 Flipkart Sales Analysis Dashboard (Power BI)

## 🚀 Overview
This project showcases an **interactive Power BI dashboard** built to analyze Flipkart sales data and extract meaningful business insights.

## 📌 Key Features
- KPI Metrics: **Total Sales, Avg Sales, Avg Rating, Quantity Sold**
- **Sales Trend Over Time**
- **Top Product Categories Analysis**
- **Region-wise Sales Distribution (Global Map)**
- **Payment Method Analysis** (COD, UPI, Credit/Debit Cards, Net Banking)
- **Sales Breakdown by Region & Payment Method**
- **Quantity Sold by Product Category**
- **Sales Performance by Customer Ratings**



## 🛠 Tools & Technologies
- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  

## 🎯 Objective
The goal of this project is to:
- Understand customer purchasing behavior  
- Identify top-performing categories  
- Analyze regional sales trends  
- Explore preferred payment methods


## 📌 Conclusion
This dashboard helps in making data-driven decisions by providing clear insights into sales performance and customer trends.
